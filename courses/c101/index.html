C101 - Introduction to C Language

Greetings fellow programmers! Codeville welcomes you to C101 - Introduction to C language.

This course is essentially the first step of yours into the world of hardcore programming. The world of programming is itself very fascinating, but we will try to add some bits of our own to make is even more fun to learn.

Why C101?

The C programming language is a general purpose programming language that is used to develop anything and everything that you see around. It includes the smart phone that you have in your hand right now, the laptop or computer on which you play games or surf Internet. In short every technology that you see or use in day today life is somehow related to C language.
Through C101 you will learn the practical aspect of the C programming language, the way it is implemented and some of the real world application. C101 will give you analytical and rational thinking, problem solving approach that is today the most required qualities.

C101 will help you build 4 real world applications, namely

  1. Guess the number! - A number game
  2. Tic Tac Toe - A game we all love
  3. Game score management - Learn how you can store scores of games
  4. Chat application - Similar to any chat application you use
