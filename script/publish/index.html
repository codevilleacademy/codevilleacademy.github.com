Hello!
Thank you for contributing. To publish your script you need to fill out a simple form. Thats it! All the formating and beautification will we taken care off.
Fill out the fields below which represent different sections of the script. Once you submit the script to us, Codeville Team will verify your script, format it in the best possible way and publish it.
Cheers!


The title should be self-explanatory.
Ex: Get live scores of a cricket match
The display name of the Author of the script.
Ex: Arpit Bhayani
Profile link: your web presence. Should be your website address or even your facebook profile link is cool!
Ex: http://facebook.com/itsmearpit
Email address will be used for contact.
Comma (,) separated list of tags that you want to associate with the script.
Ex: python, live-score, cricket, notification
In few lines describe your script. This short description will be used while sharing and publishing your script.
The actual script body goes here, which will include everything you want to write about it.
Ex: script source, how to execute, and anything else that you want to include.
